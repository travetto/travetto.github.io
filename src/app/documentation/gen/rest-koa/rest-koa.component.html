Rest-Koa

KOA Provider 
$ npm install @travetto/rest-koa

The module is a koa provider for the Rest module. A valid configuration of RestApp would look like:

Wiring up Koa Provider 
export class SampleConfig {{ '{' }}

  @InjectableFactory()
  static getProvider(): RestApp {{ '{' }}
    return new KoaRestApp();
  {{ '}' }}
{{ '}' }}
