Rest-Aws-Lambda

Install: AWS Provider  
$ npm install @travetto/rest-aws-lambda

The module is an aws-serverless-express provider for the Rest module. A valid configuration of RestAppProvider would look like:

Code: Wiring up AWS Lambda Provider  
export class SampleConfig {{ '{' }}
  @InjectableFactory()
  static getProvider(): RestAppProvider {{ '{' }}
    return new RestAwsLambdaAppProvider();
  {{ '}' }}
{{ '}' }}

Default Stack

When working with express applications, the module provides what is assumed to be a sufficient set of basic filters. Specifically:

  • compression()
  • cookieParser()
  • bodyParser.json()
  • bodyParser.urlencoded()
  • bodyParser.raw({{ '{' }} type: 'image/*' {{ '}' }})
