Rest-Express

The module is an express provider for the Rest module. A valid configuration of RestAppProvider would look like:

export class SampleConfig {{ '{' }}
  @InjectableFactory()
  static getProvider(): RestAppProvider {{ '{' }}
    return new RestExpressAppProvider();
  {{ '}' }}
{{ '}' }}

Default Stack

When working with express applications, the module provides what is assumed to be a sufficient set of basic filters. Specifically:

  • compression()
  • cookieParser()
  • bodyParser.json()
  • bodyParser.urlencoded()
  • bodyParser.raw({{ '{' }} type: 'image/*' {{ '}' }})
  • session(this.config.session)
