Travetto
A Realtime Framework for Everyone

What is Travetto?

Travetto is a Typescript framework for building complex backend applications. It is a unique architecture that emphasizes:

  • Realtime Development – No need to restart your server to see changes applied
  • Realtime Testing – Testing occurs inline (with VS Code Plugin), on save, giving you instant feedback to your unit tests
  • Realtime Modeling – Modifications to data model reflect immediately, and where necessary, underlying schemas are modified in realtime to support changes
  • Typescript Focused – Leverages the rich type information (as well as some fancy ASTs) to choose smarter defaults.
  • Embracing Existing Frameworks – Utilizies existing frameworks as much as possible, to gain the benefit and maturity that each element provides.
  • Other Standard Application Featuers – Dependendy Injection, Code over Configuration, Test Framework, etc.
